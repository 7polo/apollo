#### How to Start
---
1. 根据 sql/apollo.sql 创建数据库
2. 将 application-demo.yml 文件中配置完善， 并将之复制到 application.yml 中
3. 默认账号和密码： admin/admin

可通过 SpringBoot 的启动方式启动或打包
